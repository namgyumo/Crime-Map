# Crime-Map

부산진구의 범죄자 및 재발 범죄자 위치, 여성 1인 사업장, 여성 안심 귀갓길 등 범죄 주의 지역을 지도에 표시하고, 현재 위치를 기반으로 주변 주의 지점을 알려주는 프로젝트입니다.

## 주요 기능

- 부산진구 내 범죄자 및 재발 범죄자 위치 표시
- 여성 1인 사업장, 여성 안심 귀갓길 등 범죄 주의 지역 지도 표시
- 현재 위치 기반 주변 주의 지점 안내
